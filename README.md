# Checkmate
### Master Your Day, Every Day.

![Checkmate Preview](https://res.cloudinary.com/ddzreu2to/image/upload/v1769528073/vecteezy_classic-chess-horse-vectors-art_67945655-removebg-preview_nc3y9f.png)

Checkmate is a premium, high-performance personal productivity suite designed to help you organize habits, tasks, goals, and schedules in one seamless, beautiful interface. Built with a focus on aesthetics and fluid user experience, Checkmate leverages modern web technologies to deliver a native-app feel on the web.

## ‚ú® Key Features

*   **Glassmorphic Design**: A stunning, modern UI featuring high-quality glassmorphism, smooth gradients, and vibrant neon accents.
*   **Fluid Animations**: Every interaction is animated using Framer Motion for a buttery-smooth feel (60fps).
*   **Smart Dashboard**: At-a-glance view of your daily streaks, tasks, and punctuality scores.
*   **Habit Tracking**: Build lasting habits with visual streak counters and satisfaction-inducing check animations.
*   **Task Management**: prioritize your day with High, Medium, and Low priority sorting and "Add New" quick actions.
*   **Goal Setting**: Track long-term progress with visual bars and milestone tracking.
*   **Responsive**: Fully optimized for Desktop, Tablet, and Mobile experiences.
*   **Privacy First**: 100% local storage. Your data never leaves your device.

## üöÄ Getting Started

Follow these steps to get Checkmate running locally on your machine.

### Prerequisites

*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/yourusername/checkmate.git
    cd checkmate
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  **Open in Browser**
    Navigate to `http://localhost:8080` (or the port shown in your terminal).
